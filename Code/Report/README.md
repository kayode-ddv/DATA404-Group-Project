# Affordable Care Act Health Insurance Marketplace

This project analyzes and predicts instability in the Affordable Care Act (ACA) marketplace across U.S. states using a multimodal data approach.

We combine economic, policy, and insurance market data to better understand where and why healthcare access risk may occur.

## Why Should This Matter?

This project focuses on identifying early warning signs of healthcare access risk in underserved communities.

Our goal is to support proactive decision-making that can help policymakers and communities respond earlier and more effectively.

## Data Sources

- Centers for Medicare & Medicaid Services (CMS)
- American Community Survey (ACS)
- Kaiser Family Foundation (KFF)
- Bureau of Labor Statistics (BLS)

## Methods

We applied machine learning models including:

- Logistic Regression
- Decision Trees
- Random Forest
- Gradient Boosting

We also used cross-validation and hyperparameter tuning to improve model performance and reduce overfitting.
